📘 NotebookLM 슬라이드 프롬프트 가이드
✨ 교재 소스 → 이미지 프롬프트 → 슬라이드 키프레임

NotebookLM으로
20개 예시 이미지를 슬라이드로 만들기

NotebookLM에 교재를 소스로 추가한 뒤, 아래 프롬프트를 채팅창에 복사하여 붙여넣으면 교재의 이미지 생성 사례를 바탕으로 영상용 키프레임 슬라이드를 만들 수 있습니다.

✅ 사용 흐름

  1. NotebookLM에 교재 PDF 또는 문서를 소스로 추가합니다.
  2. 교재 안의 Part 2 이미지 생성 프롬프트 사례집을 기준으로 작업합니다.
  3. 프롬프트 1을 먼저 입력하여 20개의 키프레임 이미지 프롬프트를 만듭니다.
  4. 프롬프트 2를 입력하여 20개의 프롬프트를 슬라이드로 시각화합니다.
  5. 각 슬라이드는 영상 제작에 사용할 수 있는 장면 이미지 역할을 합니다.

🎬 핵심 목적

이 방법은 교재 내용을 단순 요약하는 것이 아니라, 교재 속 이미지 생성 사례를 바탕으로 영상 제작에 필요한 20개의 장면 이미지를 한 번에 구성하기 위한 실습입니다.

권장 순서: 프롬프트 1 실행 → 결과 확인 → 프롬프트 2 실행

프롬프트 1 키프레임 이미지 프롬프트 만들기

제공된 소스의 part 2 AI 이미지 생성 프롬프트 사례집 1번-20번까지의
이미지생성 프롬프트를 기반으로 영상을 만들 거야.
20개의 키프레임 이미지를 만들기 위한 프롬프트를 만들어줘.

프롬프트 2 슬라이드 시각화 만들기

1번부터 20번까지의 프롬프트를 시각화해서 슬라이드를 만들어줘.
영상을 만들 수 있도록 각각의 슬라이드가 키프레임이 되도록 해주고,
슬라이드에 텍스트는 넣지 말아줘.

📌 수강생 안내 문구

NotebookLM은 업로드한 교재를 기준으로 답변을 만들어주는 AI 학습 도구입니다. 따라서 먼저 교재를 소스로 추가해야 하며, 그다음 위의 프롬프트를 붙여넣으면 교재 내용과 연결된 이미지 프롬프트와 슬라이드 구성을 만들 수 있습니다.
